Cursor

Cette composante permet d'avoir un curseur personnalisé.

Comment l'utiliser

Afin d'utiliser la composante, mettre l'attribut data-component="Cursor" sur une structure html en dessous de la balise footer.

exemple de l'emplacement

Pour personnaliser le style du curseur, il faut essentiellement créer un fichier cursor.scss et ajouter les modifications nécéssaires
